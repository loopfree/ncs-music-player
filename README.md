# NCS Music Player
> This is a music player WebApp which play free music from NCS (No Copy Right Sounds).

# Build with
* HTML
* Tailwind CSS
* Vue.js

# Getting Started

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

# Deployment
Published to `https://pleasant-match.surge.sh/`
